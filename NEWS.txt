What's new in utx 0.4?

 * migrated to Python3
 * migrated from py2exe to pyinstaller


What's new in utx 0.3?

 * support multiple source arguments
 * "-m" option to define reaction to "no match"
 * return exit code on errors
 * [cpx] much faster copying
 * [cpx] "update" mode (similar to "cp -u")
 * various fixes


What's new in utx 0.2?

 * Exclude patterns with -x or -X
 * Exit nicely on ^C
 * [cpx] report progress for individual files
 * [cpx] create destination directory if missing
 * Fixed bug in the implicit (directory, pattern) computation
